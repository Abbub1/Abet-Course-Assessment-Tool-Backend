The PostSurvey Function.

This function creates and then adds it to the database It is an async Task to pass exceptions to the Controllers.SurveyController in Controllers It first checks for null inputs in all survey subfields. It then formats term and EUID to a standard. Finally, it checks if the survey already exists, and if not, adds and saves it.
